1.1 Emacsのorg-modeで論文を書く（その5：htmlへのexportのいろいろ）   emacs orgmode html export css color

学会発表や論文作成にあたっては，当然のことながら，その分野の他の研究者の論文を読んでまとめるなどの作業を行う．そこで，論文の要旨などをorg-modeにざっとまとめておくと，pdfにもhtmlにもexport出来て便利である．pdfは印刷に向いているが，htmlは多くの論文をいっぺんに見るのに向いており，また，compileの時間もpdfより圧倒的に速い．そこで，今回は，org-modeからhtmlへexportする際の有用な小技について書いてみたい．

1.1.1 フォントの色の変更やハイライト

  1. Org Macros

    • フォントの色を変更する方法はいろいろあるが，ハイライトや背景の色の変更までできるこの Org Macrosが一番便利である．リンク先からダウンロードして，適当なところに保存し，解凍しておく．ここでは，/Users/taipapa/hoge/fuga/org-macros.setupに置くことにする．使い方は簡単で上記のwebsiteに書いてあるとおり，各org fileの先頭に以下のように記述してorg-macros.setupの場所を教えてやれば良い． 

      #+INCLUDE: /Users/taipapa/hoge/fuga/org-macros.setup

      これだけである．

    • いくつか使い方の例をあげておく
    • 赤になるかな
      • 赤くなるかな？
      • ボールドで赤くなるかな？
      • 黄色にハイライトされるかな？
      • 黄色にハイライトされて文字はボールドになるかな？
      • 背景が黄色になるかな？
      • 背景が黄色になって文字はボールドになるかな？
